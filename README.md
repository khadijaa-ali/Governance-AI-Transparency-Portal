Description

This project is an AI-powered system designed to analyze government procurement data, detect anomalies, and visualize public spending patterns. It also includes a citizen chatbot and a dashboard for interactive insights.

Features
Data Generation
Synthetic government procurement dataset
Department-wise budget simulation
Realistic corruption patterns injected
Feature Engineering
Log transformation of amounts
Department and company relationship validation
Time-based features (year, month, day)
Project size classification
Days since latest project calculation
Machine Learning
Random Forest Classifier
XGBoost Classifier
Model evaluation using multiple metrics
ROC-AUC based model selection
Model saved using joblib
Anomaly Detection
Rule-based anomaly labeling
Isolation Forest for unsupervised detection
Detection of:
Wrong companies
Extreme budget values
Delayed or inconsistent projects
Visualization
Department-wise spending analysis
Time series budget trends
Anomaly distribution charts
Feature importance visualization
Streamlit App
Government budget dashboard
Interactive filters (year, department)
Real-time anomaly visualization
Citizen chatbot system
CNIC verification module
Utility bill inquiry system
English and Urdu support
Tech Stack

Python, Pandas, NumPy, Scikit-learn, XGBoost, Streamlit, Plotly, Matplotlib, Seaborn

Output
AI-based anomaly detection system
Interactive government transparency dashboard
Citizen service chatbot
